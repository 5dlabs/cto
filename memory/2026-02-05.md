## Server Routes

| Route | Method | Purpose |
|-------|--------|---------|
| `/health` | GET | Health check |
| `/oauth/start?agent=<name>` | GET | Start OAuth flow → redirects to Linear |
| `/oauth/callback` | GET | OAuth callback from Linear |
| `/oauth/refresh/<agent>` | POST | Refresh access token |

## OAuth Flow

1. `GET /oauth/start?agent=morgan` → redirects to Linear
2. User authorizes at Linear (with `actor=app`)
3. Linear redirects to `/oauth/callback?code=...&state=morgan`
4. Server exchanges code for tokens
5. Tokens saved to `~/.config/cto-lite/pm-lite.json`
