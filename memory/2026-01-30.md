# CTO Lite Session Memory - 2026-01-30

## Project Overview
CTO Lite is a freemium desktop app (Tauri + React) that runs the CTO platform on a local Kind cluster. Target: individual developers without enterprise infrastructure.

## Key Decisions Made This Session

### Architecture
- **Cloudflare tunnels**: User runs own `cloudflared`, connects via OAuth to their Cloudflare account - zero cost to us at scale
- **GitHub**: Shared GitHub App with `repository_hooks:write` permission, user installs on their repos
- **Storage**: SQLite for local state (not JSON files)
- **MCP**: Host daemon managed by Tauri (not in-cluster)
- **Controller**: Reuse as-is (Linear integration is optional/nullable)
- **PM**: Fork to `pm-lite` (Linear coupling too deep to conditionally disable)

### File Boundaries (STRICT)
**ALLOWED** (CTO Lite agent only):
- `crates/cto-lite/` - All Lite code

- `docs/cto-lite.md`
