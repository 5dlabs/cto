## Solana Node Hosting Requirements

For a Solana validator/RPC node:
- **CPU**: 32+ cores (high single-thread performance)
- **RAM**: 128GB+ minimum
- **Storage**: 1.5TB+ NVMe SSD
- **Network**: 1Gbps+ symmetric (critical for consensus)
- **Bare metal recommended** (no virtualization overhead)

### Budget Options for Solana Node
1. **Hetzner CCX62** - ~€500/mo, excellent bare metal
2. **Scaleway Dedibox** - Competitive EU pricing
3. **Oracle Cloud Free Tier** - ARM-based, good for testnet
