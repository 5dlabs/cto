Agent Routing Flow

Morgan determines the agentHint for each task during intake. The play workflow routes to the correct implementation agent, then all support agents run sequentially.

Task-Based Agent Routing

During intake, Morgan assigns an agentHint to each task based on explicit mentions, dependency inheritance, or keyword inference. Task 1 is always forced to Bolt, but Bolt can also handle additional infrastructure tasks throughout the project.
