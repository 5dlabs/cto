## Platform Overview

The CTO platform orchestrates multiple specialized AI agents:

| Agent | Role | Description |
|-------|------|-------------|
| **Atlas** | Orchestrator | Coordinates workflows and manages agent lifecycles |
| **Bolt** | Builder | Generates and deploys applications |
| **Rex** | Backend Dev | Implements backend services and APIs |
| **Blaze** | Frontend Dev | Creates modern UI/UX with React and shadcn |
| **Morgan** | Project Manager | Manages GitHub Projects and task tracking |
| **Cipher** | Security | Handles security scanning and remediation |
| **Tess** | QA | Performs testing and quality assurance |
