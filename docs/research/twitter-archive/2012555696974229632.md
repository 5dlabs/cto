# Research: GitHub Copilot CLI

## Original Tweet

> Have you tried /delegate yet in Copilot CLI? Watch the magic 

**Author**: @github (github)  
**Posted**: January 17, 2026 at 04:00 PM

## Analysis

GitHub Copilot CLI's /delegate feature represents a valuable pattern for our multi-agent platform: natural language task delegation to specialized agents. This is directly applicable to our orchestration layer. We could implement: (1) A similar CLI interface for delegating tasks to Rex, Blaze, Tess, etc. using natural language, (2) An intelligent routing system that parses user intent and selects the appropriate agent(s), (3) A conversational workflow where users describe what they want built and the system orchestrates multiple agents automatically. The technical fit is strong - we already have the agent infrastructure, we'd be adding a more intuitive delegation interface. This would significantly improve developer experience and reduce the learning curve for using our platform. The implementation would involve building a natural language parser (possibly using LLM-based intent classification), a task routing engine, and a CLI tool in Rust. Moderate effort (2-3 weeks) but high impact on usability.

**Implementation Potential**: 🔍 Worth Investigating (0.82)

### 💡 Implementation Ideas

- Build a Rust-based CLI tool with /delegate command that routes tasks to appropriate agents (Rex for architecture, Blaze for implementation, Tess for testing)
- Implement natural language intent parser using LLM to understand user requests and map to agent capabilities
- Create an agent routing engine that analyzes task complexity and automatically selects single or multi-agent workflows
- Add conversational feedback loop where agents can ask clarifying questions before executing
- Integrate with existing Argo Workflows to translate natural language requests into workflow definitions
- Build a task decomposition system that breaks complex requests into sub-tasks for different agents
- Create MCP tool for agent capability discovery and matching based on natural language descriptions
