# Research: execution-tracing

## Original Tweet

> "What happens when I click this button?"
> 
> Tidewave can now answer this visually.
> 
> Point → Ask → Get a Mermaid diagram of the execution flow through your stack.

**Author**: @hugobarauna (hugobarauna)  
**Posted**: January 16, 2026 at 04:58 PM

## Analysis

This tweet describes a highly valuable feature for our multi-agent development platform: visual execution flow tracing. The ability to trace 'what happens when X executes' and generate Mermaid diagrams would be extremely useful for debugging agent workflows, understanding Argo Workflow execution paths, and visualizing MCP tool call chains. We could implement this as: (1) A workflow visualization tool that traces Argo Workflow executions and generates Mermaid diagrams, (2) An agent action tracer that shows the execution flow when an agent performs a task, (3) A debugging MCP tool that agents can use to understand their own execution context. The technical fit is strong - we already use workflow orchestration, and Mermaid diagram generation is straightforward. This would significantly improve observability and debugging capabilities. The main implementation effort would be in instrumenting our execution paths and building the trace-to-diagram converter. Tidewave appears to be a specific product, so we'd be building our own version rather than integrating theirs.

**Implementation Potential**: 🔍 Worth Investigating (0.85)

### 💡 Implementation Ideas

- Build an Argo Workflow execution tracer that generates Mermaid sequence diagrams showing the flow through workflow steps
- Create an MCP tool that agents can call to visualize their current execution context and call chain
- Implement agent action tracing that captures when Rex calls Blaze, when Tess validates, etc., and renders as a Mermaid diagram
- Add a debugging dashboard that shows 'what happened when this agent action was triggered' with visual flow diagrams
- Build a Rust library for execution tracing that can be embedded in our agent runtime
- Create a workflow step that automatically generates execution flow diagrams for complex multi-agent tasks
- Implement a 'explain this workflow' feature that uses execution traces to generate documentation with Mermaid diagrams
