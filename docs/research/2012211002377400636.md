# Research: prompt engineering

## Original Tweet

> Advice for anyone prompting AI agents:
> 
> The classic coding books contain some of the best descriptions of "good code" ever written.
> 
> Use them in your prompts, skills, and AGENTS​.md files

**Author**: @mattpocockuk (mattpocockuk)  
**Posted**: January 16, 2026 at 05:11 PM

## Analysis

This tweet suggests a concrete implementation pattern: incorporating classic coding principles from established books (Clean Code, Design Patterns, Refactoring, etc.) into agent prompts and configuration files. We could implement this by: 1) Creating a curated knowledge base of coding principles extracted from canonical sources, 2) Building a system to inject relevant principles into agent system prompts based on task context, 3) Enhancing our AGENTS.md or similar config files with structured coding standards, 4) Developing a 'coding standards' MCP tool that agents can query. The value is high because better-defined coding standards would directly improve code quality from our AI agents. This is actionable and specific enough to prototype, though we'd need to research which principles to extract and how to structure them for LLM consumption.

**Implementation Potential**: 🔍 Worth Investigating (0.82)

### 💡 Implementation Ideas

- Build a 'coding-standards' MCP tool that serves canonical programming principles from Clean Code, SOLID, etc. that agents can query contextually
- Create structured AGENTS.md templates that include sections for coding principles, with examples from classic books
- Develop a knowledge base of coding patterns/anti-patterns extracted from canonical sources, injected into agent system prompts based on language/framework
- Implement a prompt enhancement layer that automatically adds relevant coding principles to agent tasks (e.g., 'refactoring' tasks get Fowler's principles)
- Create agent 'skills' files that codify best practices from classic books as reusable, queryable knowledge
- Build a linting/validation system that checks agent output against principles from canonical sources
