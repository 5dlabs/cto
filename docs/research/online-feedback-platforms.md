# Online Feedback Platforms Research

## Top Platforms Overview

### Behavioral/UX Analysis
- **Hotjar** - Heatmaps, session recordings, surveys
- **FullSession** - Session replay and UX analytics
- **Mixpanel** - Product analytics with feedback features

### Feature Requests & Voting
- **Canny** - Feature voting, roadmap management, changelog
- **ProductLift** - Open-source alternative to Canny
- **FeatBit** - Feature flags with feedback

### Survey Tools
- **SurveyMonkey** - Comprehensive survey creation & distribution
- **Typeform** - Interactive, conversational forms
- **Google Forms** - Free, simple survey tool
- **JotForm** - Advanced form builder

### NPS/CSAT Measurement
- **Delighted** - Simple customer feedback collection
- **Wootric** - In-app surveys and NPS scores
- **Satismeter** - In-app feedback and surveys
- **Retently** - NPS and customer satisfaction

### Enterprise Solutions
- **Qualtrics** - Enterprise research and feedback management
- **Medallia** - Customer experience management (CXM)
- **GetFeedback** - Salesforce-integrated feedback
- **Medallia** - Real-time CX insights

### Integrated/Embedded
- **Intercom** - Messenger with feedback widgets
- **Zendesk** - Support with feedback features
- **Freshdesk** - Helpdesk with feedback tools
- **UserGuiding** - In-app guidance and feedback collection

## Recommendations by Use Case

| Use Case | Recommended Tools |
|----------|-------------------|
| UX/Heatmaps | Hotjar, FullStory |
| Feature Prioritization | Canny, ProductLift |
| Quick Surveys | Typeform, Google Forms |
| NPS Tracking | Delighted, Wootric |
| Enterprise CX | Qualtrics, Medallia |
| Embedded Feedback | UserGuiding, Intercom |
