# Research: code review

## Original Tweet

> As agents get better, the pressure shifts to the ends of the workflow, the context and intent — and reviews.
> 
> We're introducing Linear Code Reviews as native part of the platform, designed for humans and agents alike.

**Author**: @karrisaarinen (karrisaarinen)  
**Posted**: January 20, 2026 at 04:14 PM

## Analysis

This tweet announces Linear's code review feature designed for both humans and agents. While it's a product announcement rather than technical implementation details, it highlights a critical gap in our platform: **native code review workflows for AI agents**. Currently, our agents (Rex, Blaze, Tess) generate code but lack a structured review process. We could implement: (1) A review orchestration layer in our Argo Workflows that pauses for human/agent review before merging, (2) MCP tools for code review operations (comment, approve, request changes), (3) An agent-specific reviewer (potentially a new agent role) that performs automated code quality checks, security scans, and best practice validation, (4) Integration patterns with existing review systems (GitHub PR reviews, GitLab MRs) that our agents can interact with programmatically. The 'designed for agents' aspect is particularly relevant - we need review interfaces that agents can parse and respond to, not just human-readable comments. The score is 0.85 rather than 0.9+ because this is an announcement without implementation details, but the concept directly addresses a real need in agent-driven development workflows.

**Implementation Potential**: 🔍 Worth Investigating (0.85)

### 💡 Implementation Ideas

- Build a 'ReviewAgent' that performs automated code review using LLM analysis before human review
- Create MCP tools for code review operations: mcp-review-comment, mcp-review-approve, mcp-review-request-changes
- Implement Argo Workflow templates with review gates that pause execution pending approval
- Design agent-parseable review feedback format (structured JSON) alongside human-readable comments
- Integrate Linear API for bidirectional sync of code review status with our platform
- Build Kubernetes CRD for CodeReview resources that track review state across agent workflows
- Create review quality metrics dashboard showing agent vs human review patterns
