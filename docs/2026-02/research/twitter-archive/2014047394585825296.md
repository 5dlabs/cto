# Research: prompt_management

## Original Tweet

> Did you know you can stash your prompt in Claude Code? 
> 
> Ctrl+S saves your draft. Allows you to send a different prompt. Your saved draft is then auto-restored.
> 
> No more copying to Notes. It's like git stash, but for your prompts.

**Author**: @adocomplete (adocomplete)  
**Posted**: January 21, 2026 at 06:48 PM

## Analysis

This describes a valuable UX pattern for prompt management that we could implement across our agent interaction interfaces. The 'git stash for prompts' concept is highly actionable - we could build a prompt draft system that allows developers to save work-in-progress prompts when they need to quickly send a different query to an agent, then auto-restore the original. This would improve developer workflow when iterating on complex agent instructions. Implementation would involve: 1) Adding draft storage (Redis/local state) for prompt text, 2) Keyboard shortcuts (Ctrl+S) in our web UI, 3) Auto-restore logic when returning to an agent conversation, 4) Optional: versioning/history of saved drafts. This directly benefits Rex, Blaze, and other agents by reducing friction in the prompt refinement process. Medium effort (1-2 weeks) but high impact on developer experience.

**Implementation Potential**: 🔍 Worth Investigating (0.82)

### 💡 Implementation Ideas

- Build a prompt draft system with Ctrl+S keyboard shortcut in React UI to save work-in-progress prompts
- Implement auto-restore functionality that brings back saved drafts when user returns to agent conversation
- Add Redis-backed storage for prompt drafts with user session management
- Create a 'stash list' UI component showing all saved prompt drafts with timestamps
- Implement prompt versioning to allow multiple saved drafts per agent conversation
- Add visual indicator when a draft is saved (similar to git stash confirmation)
- Build keyboard shortcut system (Ctrl+S save, Ctrl+Shift+S restore) for power users
- Extend to CLI interface for developers who prefer terminal-based agent interaction
