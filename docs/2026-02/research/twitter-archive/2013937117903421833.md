# Research: MCP Server

## Original Tweet

> Google just released its own MCP (Model Control Plane) server for Stitch, turning LLM coding agents into UI-savvy collaborators.
> 
> The new Stitch MCP Server pipes designs directly into your IDE.
> 
>  Generate new screens
>  Fetch code from any design
>  Give your agent full visual

**Author**: @WesRoth (WesRoth)  
**Posted**: January 21, 2026 at 11:30 AM

## Analysis

This is highly actionable for our platform. Google's Stitch MCP Server represents a concrete integration opportunity that could significantly enhance our agents' capabilities. We could: 1) Integrate the Stitch MCP Server as a new tool for our agents (Rex, Blaze, Tess) to consume design specifications and generate UI code, 2) Build a workflow that pipes Figma/design tool outputs through Stitch into our agent orchestration pipeline, 3) Create a specialized UI generation agent that leverages this MCP server. The value proposition is clear: our agents currently lack visual design context, and this would enable them to generate frontend code from actual designs rather than just specifications. Since we already use MCP tools, integration should be straightforward. The main uncertainty is implementation details - we need to understand the API, data formats, and whether it works with our existing MCP infrastructure.

**Implementation Potential**: 🔍 Worth Investigating (0.85)

### 💡 Implementation Ideas

- Integrate Stitch MCP Server as a new tool in our MCP tools registry for agent consumption
- Build a design-to-code workflow in Argo that uses Stitch to convert design files into implementation tasks
- Create a specialized 'UI Agent' that combines Stitch MCP with our existing coding agents for frontend generation
- Add Stitch integration to our IDE tooling so agents can fetch design context during development
- Build a Kubernetes operator that manages Stitch MCP Server instances for multi-tenant agent workloads
- Develop a Rust client library for Stitch MCP to integrate with our existing Rust-based infrastructure
