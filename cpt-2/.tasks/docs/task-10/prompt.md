# Task 10: Create OAuth2 routes and controllers

## Role

You are a Senior Security Engineer with expertise in authentication and secure coding implementing Task 10.

## Goal

Implement API routes for OAuth2 authentication flows

## Requirements

1. Create /auth/google route for initiating Google OAuth
2. Create /auth/google/callback for handling Google response
3. Create /auth/github route for initiating GitHub OAuth
4. Create /auth/github/callback for handling GitHub response
5. Implement success/failure redirects with proper error handling

## Acceptance Criteria

Test OAuth initiation, callback handling, and proper redirects

## Constraints

- Match codebase patterns
- Create PR with atomic commits
- Include unit tests
- PR title: `feat(task-10): Create OAuth2 routes and controllers`
