# CTO Parallel Test Repository

This repository is used for E2E testing of the CTO Platform's parallel agent workflows.

## Purpose

This test repository allows us to validate:
- Agent workflow execution (Rex, Cleo, Blaze)
- Parallel task processing
- GitHub integration
- Artifact generation and management

## Structure

```
cto-parallel-test/
├── cto-config.json           # CTO Platform configuration
├── docs/
│   └── .taskmaster/
│       └── docs/
│           └── test-prd.txt   # Test PRD document
├── src/                       # Source code (generated by agents)
└── tests/                     # Test files (generated by agents)
```

## Usage

This repository is automatically managed by E2E test scripts in the main `cto` repository.

### Reset Environment

```bash
# From the cto repository root
make e2e-reset
```

## Configuration

The `cto-config.json` file configures which agents are enabled for this test project.
