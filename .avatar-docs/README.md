# Avatar Docs

Local catalog for Morgan avatar options: talking portrait / talking-head
video, 3D generation, rigging, puppeteering, UI/runtime glue, subscribed-model
candidates, and provider/credit contingencies.

The active execution plan lives in `docs/avatar/`; this directory is the
research database of what is available.

## Layout

| Path | Contents |
| --- | --- |
| `available-models.md` | Model catalog covering open-source and paid candidates by pipeline stage. |
| `provider-credit-map.md` | Provider, startup-credit, GPU, serverless, and 3D API contingency map. |
| `index/` | One compact JSON summary per repo plus `scenario-models.json`. |
| `sources/repos.json` | Manifest used to clone and classify the repos. |
| `pipeline-shortlist/` | Peer-reviewed runtime/provisioning shortlist, archive notes, and next research command. |
| `huggingface/` | Recent Hugging Face model/Space catalog grouped by pipeline stage. |
| `repos/` | Local shallow clones, named `<owner>__<repo>`. This is a large local cache and is intentionally gitignored. |

## Short take

The clearest near-term Plan B remains a high-quality **talking portrait** pipeline from a cleaned Morgan image plus audio: FantasyTalking, PersonaLive, LivePortrait, MuseTalk, EchoMimic, SadTalker, Wav2Lip/LatentSync, and related repos. The likely deeper path is **3D model + rigging + puppeteering**: UniRig/SkinTokens for rigging, AniGen/Tripo-family repos for animatable or generated 3D assets, MakeHuman/Arc2Avatar/AvatarCLIP/DRiVEAvatar-style projects for human/avatar references, and Gradio/Tripo MCP or Scenario apps for orchestration.
