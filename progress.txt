## Codebase Patterns
- Intake workflow requires Morgan OAuth app to be installed in Linear workspace for auto-assign
- `auto_assign_morgan` only works if Morgan user exists in Linear workspace users search
- CLI mode JSON parsing: use `text.contains("```json")` not `text.find("```json")` to handle prefill
- API mode for intake requires ANTHROPIC_API_KEY and may timeout on large PRDs (>20k chars)
- Linear issues created: CTOPA-792, CTOPA-793, CTOPA-794 for alerthub-e2e-test
- Empty tasks.json indicates intake didn't complete, not that there are no tasks
- When AI wraps JSON in markdown code blocks, extract from code block BEFORE brace-matching
- cto_play() requires `model` parameter explicitly when `defaults.play.model` not set in config
- cto_play() requires `docs_project_directory` when tasks.json is not in default `docs/` folder
