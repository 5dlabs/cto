# Task 3: Build Web Console Frontend (Next.js/React)

## Role

You are a Senior Frontend Engineer with expertise in React, TypeScript, and modern UI/UX implementing Task 3.

## Goal

Create React-based web interface for viewing and managing notifications with modern UI components

## Requirements

1. Initialize Next.js 14+ project with App Router, React 18, shadcn/ui, TailwindCSS
2. Setup TanStack Query for server state management and React Hook Form
3. Create pages: / (dashboard), /notifications/:id (detail), /settings
4. Implement components: NotificationList (paginated), NotificationCard, CreateNotificationForm, ThemeToggle
5. Add dark/light theme support with theme persistence
6. Implement real-time polling (5s interval) for notification updates
7. Add toast notifications for user actions
8. Style with TailwindCSS and shadcn/ui components

## Acceptance Criteria

Component testing with React Testing Library, E2E tests with Playwright for user flows, accessibility testing, responsive design validation

## Constraints

- Match codebase patterns
- Create PR with atomic commits
- Include unit tests
- PR title: `feat(task-3): Build Web Console Frontend (Next.js/React)`
