## Description
Create new Next.js 15 project using create-next-app with App Router, configure TypeScript, and set up basic project structure with necessary configuration files.

## Dependencies

## Implementation Details
Run `npx create-next-app@latest --typescript --tailwind --eslint --app --src-dir --import-alias '@/*'` to initialize the project. Configure next.config.js for App Router, update tsconfig.json with proper TypeScript settings, and create basic folder structure including components, lib, and types directories.
