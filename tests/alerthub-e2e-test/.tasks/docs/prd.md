AlertHub is a multi-platform notification system with: Rex (Rust/Axum) for notification routing, Grizz (Go/gRPC) for admin API, Nova (Bun/Elysia+Effect) for integrations, Blaze (Next.js+Effect) for web console, Tap (Expo) for mobile, Spark (Electron) for desktop. Infrastructure: PostgreSQL, Redis, Kafka, MongoDB, RabbitMQ, SeaweedFS. Nova and Blaze must use Effect for type-safe error handling.
