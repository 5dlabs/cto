# `scripts/` layout

Top-level maintenance and tooling scripts are grouped under **`scripts/YYYY-MM/`** by the **month and year when each file was first added to this repository** (first “add” commit, `git log --diff-filter=A`).

The path under the month folder matches the previous flat layout, for example `scripts/2026-01/ralph-cto.sh` for what was `scripts/ralph-cto.sh`.

**Note:** This applies only to the repo-root **`scripts/`** tree. Other directories (`intake/scripts/`, `infra/scripts/`, per-package `scripts/`, etc.) are unchanged.
