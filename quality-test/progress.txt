## Codebase Patterns (For Ralph Reference)
- Use `cargo fmt --all --check` before committing
- Use `cargo clippy --all-targets -- -D warnings -W clippy::pedantic` for full linting
- Run `cargo test` to verify no regressions
- Pre-commit hooks: `pre-commit run --all-files`
- PRs should target `develop` branch (not `main`)

## Quality Fix Guidelines
1. **unused_self**: Either use self or convert to associated function `Self::method()`
2. **unnecessary_wraps**: Remove Result wrapper if not needed, or document why
3. **unused_async**: Remove async keyword if no await, or add appropriate await
4. **cast_***: Use TryFrom/TryInto with proper error handling, or document safety
5. **too_many_arguments**: Consider builder pattern or config struct
6. **too_many_lines**: Extract helper functions for logical sections
7. **disallowed_macros** (MCP): Document reason - MCP protocol requires stdout

## Build Commands
```bash
# Format check
cargo fmt --all --check

# Basic Clippy (CI)
cargo clippy --all-targets -- -D warnings

# Pedantic Clippy (quality)
cargo clippy --all-targets -- -D warnings -W clippy::pedantic

# Tests
cargo test

# Pre-commit
pre-commit run --all-files
```
